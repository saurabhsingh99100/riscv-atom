RISC-V Atom Documentation
======================================
Atom is a 32-bit embedded-class processor based on the [RISC-V](https://riscv.org/) 
Instruction Set Architecture (ISA). It is designed to be light on hardware resources 
and still powerful enough to be deployed in most embedded class applications.
